Welcome

This website explains SDG 7 (Affordable and Clean Energy) in a simple way for students. I made this website as part of my school project and I used basic HTML and CSS.

What is SDG 7?

SDG 7 is a global goal that focuses on giving everyone access to reliable, clean and affordable energy. It also encourages renewable sources like solar and wind.
